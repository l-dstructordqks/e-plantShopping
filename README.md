## Paradise Nursery Shopping — React Final Project

This project is the final assignment for the IBM Developing Front-End Apps with React course.

# Overview

Paradise Nursery Shopping (e-plantShopping) is a front-end e-commerce application where users can browse and purchase a variety of plants. The app provides a dynamic shopping experience with real-time cart updates and intuitive state management.

✨ Features
- 🌿 Browse a selection of plants
- 🛒 Add products to the shopping cart
- 🔄 Real-time cart updates
- ❌ Remove items from the cart
- ✏️ Update product quantities
⚙️ Technologies Used
- React (Functional Components and Hooks)
- Redux (State Management)
    Actions
    Reducers
    Store
